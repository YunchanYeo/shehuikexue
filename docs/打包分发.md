# 打包与直接分发（无需下载页、无需服务器）

把安装包发给用户即可，**不用**网站、**不用**自己开服务器。

---

## 发给用户的文件

| 系统 | 文件（构建后路径） | 用户怎么用 |
|------|-------------------|------------|
| **macOS** | `dist/SpeechEval.dmg` | 双击 → 拖入「应用程序」→ 打开 |
| **Windows** | `dist/SpeechEval-Setup.exe` | 双击 → 安装向导 → 从开始菜单打开 |

---

## 用 GitHub Actions 构建（推荐：只有 Mac 也能得到 Windows 安装包）

1. 把项目 push 到 GitHub  
2. 打开仓库 **Actions** → **Build installers** → **Run workflow**  
3. 等待约 30–60 分钟，在 **Artifacts** 下载：
   - `SpeechEval-macOS` → `.dmg`
   - `SpeechEval-Windows` → `SpeechEval-Setup.exe`

详细步骤见 **[docs/GitHub-Actions-빌드.md](GitHub-Actions-빌드.md)**（韩文说明）。

打标签 `v1.0.0` 并 push 后，还会在 **Releases** 页面自动发布两个安装包。

---

## 本地构建

### macOS（在 Mac 上）

```bash
chmod +x scripts/build_mac.sh
./scripts/build_mac.sh
```

得到：**`dist/SpeechEval.dmg`** → 用 U 盘、微信、邮件、网盘、GitHub Releases 等发给 Mac 用户。

### Windows（在 Windows 上）

1. 安装 [Inno Setup 6](https://jrsoftware.org/isdl.php)
2. 运行：

```bat
scripts\build_windows.bat
```

得到：**`dist/SpeechEval-Setup.exe`** → 发给 Windows 用户。

## 常见分发方式（任选）

- **微信 / QQ / 邮件**（注意文件大小约 1–3 GB）
- **百度网盘、阿里云盘** 等分享链接
- **GitHub Releases**：上传 dmg 与 exe，把 Release 下载链接发给用户（推荐，适合较大文件）
- **U 盘** 拷贝

无需网页、无需 Python、用户电脑不用装开发环境。

---

## 用户安装注意

**Mac：** 若提示无法打开 → 对应用 **右键 → 打开**。

**Windows：** 若 SmartScreen 拦截 → **更多信息 → 仍要运行**。

**首次评估：** 可能需联网下载语音识别模型（仅第一次）。
