安装说明

  1. 双击下载的 SpeechEval.dmg
  2. 将窗口中的 SpeechEval 拖到 应用程序 文件夹
  3. 从启动台或应用程序文件夹打开
  4. 若提示无法打开：对应用 右键 → 打开，确认一次即可

常见问题

需要联网吗？

安装不需要。首次使用「开始评估」时，可能会下载语音识别模型（约几百 MB），之后可离线使用。

支持哪些音频格式？

wav、mp3、m4a、flac、ogg 等常见格式。

Mac 和 Windows 安装包可以互相用吗？

不可以。请下载与您电脑系统对应的版本。
